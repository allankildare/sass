O que é?

Sass é um pré-processador CSS, é uma linguagem de extensão do CSS. Ao utilizar Sass podemos criar variáveis, funções, implementar lógica de programação, separar nosso CSS em módulos e muito mais.

Atualmente é um dos pré-processadores CSS mais famosos, se não o mais famoso, é totalmente compatível com CSS, aprovado pelo mercado, tem uma larga comunidade.

Criado por Natalie Weizenbaum e Hampton Catlin o Ruby Sass foi a implementação original do Sass, ou seja, o Sass vem do Ruby. Infelizmente, o Ruby Sass teve o final de sua vida decretado no dia 26 de março de 2019, mas continuou deixando seu legado.

Por que usar?

O Sass vem para tornar o CSS ainda mais dinâmico, evitando trabalho repetitivo, facilitando o seu dia-a-dia com o CSS a partir dos recursos que ele oferece.

Nesse website tudo foi estilizado pelo Sass, que tem sua extensão .scss e usei alguns recursos para facilitar isso, como o nesting, criação de variáveis, etc. Esse site foi feito com a finalidade de praticar essa linguagem e ao mesmo tempo produzir o interesse aos que visitam meu perfil no GitHub, agradeço seu interesse e espero que tenha bons estudos!
